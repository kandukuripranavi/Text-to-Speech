# Text-to-Speech Converter

This is a Python-based desktop Text-to-Speech application developed using Tkinter and pyttsx3.

## Features
- Converts user-entered text into speech
- Simple and user-friendly GUI
- Input validation for empty text
- Desktop-based Python application

## Technologies Used
- Python
- Tkinter
- pyttsx3

## How to Run
1. Install required libraries:
   pip install pyttsx3
2. Run the application:
   python tts_app.py
